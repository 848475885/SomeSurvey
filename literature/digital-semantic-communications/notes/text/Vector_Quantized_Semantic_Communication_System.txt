                                                                                                           V. C ONCLUSION
                                                                                  In this letter, we have proposed a framework of vector
                                                                               quantized semantic communication system and introduced
                                                                               a GAN-based implementation for image transmission. Our
                                                                               approach, named VQ-DeepSC, contains different scales of
                                                                               semantic embedding space and realises different scales of
                                                                               semantic features fusion at receiver with low transmission
                                                                               overheads. Specifically, by leveraging vector quantization, the
Fig. 4. MS-SSIM performance comparisons with a similar compression ratio       semantic feature vectors were compressed into a set of seman-

                                                                               which also made the model suitable for digital communication
                                                                               systems. In particular, VQ-DeepSC was found to outperform
                                                                               BPG and be comparable to DeepJSCC as measured with
                                                                               MS-SSIM. Hence, the proposed VQ-DeepSC is a promising
                                                                               candidate for image semantic communication systems.
